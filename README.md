# 🏙️ Gestió i Inventari de l'Espai Públic APP

Una aplicació web professional dissenyada per a la gestió, registre i anàlisi de l'espai públic urbà 

Creada per optimitzar la feina dels tècnics d'ajuntaments, brigades i agents de carrer, aquesta eina permet registrar incidències, geolocalitzar elements urbans (senyals, passos de vianants, mobiliari, residus), editar fotografies *in situ* i generar informes estadístics en PDF llestos per a les direccions tècniques.

> **📍 Implementació Destacada:** Dissenyada amb arquitectura multi-entitat, adaptada actualment per a l'**Ajuntament de Tortosa**, els seus barris, pedanies i polígons.

## ✨ Funcionalitats Principals

* 🗺️ **Geolocalització Avançada:** Integració completa amb Google Maps API (Roadmap i Satèl·lit). Cerca intel·ligent de carrers i detecció automàtica de coordenades via GPS.
* 📊 **Generació d'Informes (PDF):** Motor d'exportació d'informes amb gràfics interactius (Recharts), balanços de resolució, KPIs de gestió i distribució territorial.
* 📸 **Captura i Edició d'Imatges:** Integració amb la càmera del dispositiu i galeria. Inclou un **editor d'imatges natiu** per fer anotacions i dibuixar directament sobre la fotografia abans de guardar la incidència.
* ⏳ **"Màquina del Temps" (Històric):** Sistema de traçabilitat completa. Cada registre guarda un historial de modificacions amb l'opció de restaurar dades i fotografies a estats anteriors (còpies de seguretat per snapshot).
* 📝 **Gestió d'Incidències:** Classificació per tipologia (Pintura, Senyals, Pacificació, Zona Blava, Residus, etc.) i estats de conservació automàtics.
* 🤖 **Smart Text / Conclusions AI:** Mòdul de conclusions per als informes estadístics. Preparat per funcionar de forma autònoma o mitjançant redacció humana assistida.

---

## 🛠️ Stack Tecnològic

Aquest projecte està construït amb tecnologies modernes i orientades al rendiment:

* **Frontend:** React 18, TypeScript, Vite
* **Estils & UI:** Tailwind CSS, Headless UI, Heroicons
* **Mapes & Geolocalització:** Google Maps JavaScript API, Geocoding API, Places API
* **Gràfics:** Recharts
* **Desplegament:** Vercel (Preparat com a PWA / Mobile-First)
